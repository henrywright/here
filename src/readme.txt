See who's online now on your WordPress site.

== Description ==

Here makes use of the WordPress Transients API to keep a record of who's currently online. A green dot will appear next to each active user.

== Installation ==

1. Go to Plugins > Add New and search for "here"
2. Click on Install Now

You can then go ahead and activate the plugin.
